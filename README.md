# 春の雪 — 章読みの友

三島由紀夫『豊饒の海』第一巻「春の雪」の章ごとAI解説サイトです。

## 使い方

1. Anthropic API Key を取得（https://console.anthropic.com）
2. 右上に API Key を入力
3. 左のサイドバーから章を選ぶ
4. AIが日本語で解説を生成します
5. 下の入力欄から追加の質問もできます

## ローカルで動かす

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

`dist/` フォルダの中身を GitHub Pages にアップロードしてください。
